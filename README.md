# Repository Skripsi 

## Skripsi Pemetaan

### Deskripsi

Repository untuk kode skripsi pemetaan dalam ruang menggunakan ardrone

### Cara Menjalankan Kode

1. Jalankan simulasi atau hubungkan ke drone sebenarnya
2. Jalankan kode di terminal

```bash
python nama_kode.py
```
